
# Requirements
* PHP 8.4
* Laravel 12
* Composer
* Node JS
* NPM

# Components
* Tailwind CSS
* PrimeVue (with aura theme)
* Vue 3
* Bootstrap Icons
* PrimeVue forms

# Architecture
* Frontend: Vue 3
* Backend API: Laravel 12

# To start local server
* composer install
* npm install
* composer run dev
